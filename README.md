# AI Photo Studio

A local AI-powered photo restoration and image generation platform.

## Goals

- Restore old photographs
- Preserve facial identity
- Upscale images
- Replace backgrounds
- Generate AI portraits
- Automate workflows with PowerShell
- Run entirely on local hardware

## Technology

- ComfyUI
- PowerShell
- Python
- GitHub
- NVIDIA CUDA
