\# Workflow Standards



Every workflow must satisfy the following requirements before being considered complete.



\---



\## Naming



Use descriptive names.



Example:



01 - Basic 4x Upscale



\---



\## Layout



\- Logical node placement

\- Minimal wire crossings

\- Easy to read



\---



\## Documentation



Each workflow should contain a Note node describing:



\- purpose

\- inputs

\- outputs

\- important settings



\---



\## Validation



Every workflow must:



\- execute successfully

\- produce repeatable results

\- be tested using benchmark images



\---



\## Storage



Workflow JSON files are stored within the ComfyUI workflows directory.



They are organised into folders such as:



\- upscaling

\- restoration

\- portrait

\- background

\- experiments



\---



\## Version Control



Only stable workflows should be saved into the library.



Experimental workflows remain within the experiments folder until validated.
